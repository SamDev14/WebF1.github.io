Discovery Log

Week 2 - Week 16

Week 2 - Information Security & Unit Prep

I was first given an introduction to the unit. I was then told about my VM. We then worked in pairs to complete Website Awards worksheet. The practical ended with advice on good habits to build up.

Week 3 - Web Security

I Identified the information most people share on a daily basis. I then looked at how this information could be captured and used in a system. I then categorised that information. Finally I defined how this information can be processed and classified on various platforms e.g social media.

Week 4 - Reviewing Literature

I listed search terms for finding literature to review. I was advised about note taking.

Week 5 - Academic Writing

I learned about the benefits of replacing phrasal verbs. I looked at the importance of grammar. I learned about the importance of avoiding personal and subjective writing when writing a literature review. The lesson ended by revising some of our previous assignments to see if we could improve the quality of grammar with these techniques.

Week 6 - Introduction to Markup

I created a simple web page using HTML. I then uploaded it to the VM. I learned how to use the web server. I was told about the Mozilla Developer Network.

Week 7 - Reading Week

Week 8 - Separating Form and Content

I was introduced to CSS. I was shown the basics of the language.

Week 9 - Productivity and Efficiency: Editors

I used Atom and looked at ways to make it more efficient in order to increase productivity.

Week 10 - Design

In groups we arranged cards into categories to create a mockup of a navigation bar for a hotel. Each card represented a facility or activity provided by the hotel.

Week 11 - Using Images

I experimented with images in HTML and CSS. I was shown how to add images and manipulate them.

Week 12 - Website Marking Scheme

I looked at the Website Marking Scheme. I learned what needs to be included in my website.

Week 13 - Addressing the Web

I listed my favourite links using the correct HTML tags. I then created 3 new HTML files and added links to them in the original HTML file.

Week 14 - Usability

I sketched the layout for my site. I then reviewed the usability standards. I looked at media queries in CSS. Finally I looked at Web Patterns and how they help inform design.

Week 15 - Accessibility

Working in small groups, one of us simulated a handicap of some sort and the other helped them navigate the web. This allowed me to gain an understanding in making my site accessible to everyone.

Week 16 - Scalable Graphics on the Web

I used Inkscape for the first time to create a simple SVG. I then wrote an SVG using code in Atom.
